# S057: Equity Rate vs Corporate IROP

**Chapter 10: Competition, Finance, and Interest Rates** | **Figures**: Fig10.11 | **Period**: 1947–2011

## What This Measures

Shiller equity return deflated by BEA investment deflator (from S041). rreq = (dvr + delta_preq) / preq(-1), where dvr = real dividends, preq = real equity price. Compared to Ch6 S105 corporate IROP. Tests whether equity returns track the incremental rate of profit on corporate capital.

## Construction

1. Load S041 for Shiller+deflator
2. Compute real equity return
3. Load S105 IROP
4. Align
