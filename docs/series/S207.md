# S207: Wage Equivalent and Corporate/Noncorporate Split

**Chapter 6: Capital and Profit** | **Period**: 1947–2024

## From the Book

> Wage equivalent (WEQ2) and corporate/noncorporate split. NIPA T1.13, T1.14.
>
> — Shaikh (2016), Appendix 6.8.I.2


## Construction

| Step | Operation | Description |
|------|-----------|-------------|
| 1 | `load` | Load BEA T1.13 (proprietors' income), T1.14 (corporate accounts) |
| 2 | `calculate` | Calculate corporate wage-profit ratio sigma = ECcorp/Pcorp |
| 3 | `calculate` | Calculate WEQ2 = (sigma*PropInc - ECprop)/(1+sigma) |
| 4 | `calculate` | Noncorporate profit = PropInc - WEQ2 |
