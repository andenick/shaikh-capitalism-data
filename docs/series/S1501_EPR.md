## 1. Why this series is extendable

S1501 is a `time_series` series (per registry `content_type`). Both segments — MeasuringWorth USCPI (1774-2011) and BLS CPI-U (1913-present) — are continuously published. The post-1913 portion of MeasuringWorth IS BLS CPI-U, so the extension is a direct continuation of the same time series Shaikh used.

## 2. Construction classification

S1501 is `direct`, not `formula`. The lazy-splice prohibition (Anu rule) is satisfied because we are appending the same observed BLS CPI-U index, not a growth rate computed on a derivative quantity.

## 3. Method

```
INPUTS
  S1501_book      = MeasuringWorth USCPI 1774-2011, base 1982-84 = 100
  FRED_CPIAUCNS   = BLS CPI-U All Urban Consumers (NSA), annual avg, base 1982-84 = 100

OVERLAP CHECK
  overlap_year = 2011
  Assert: S1501_book[2011] and FRED_CPIAUCNS[2011] both non-NaN
  Walk back through 2010, 2009, ... 2005 if needed.

REINDEX
  scale_factor = S1501_book[overlap_year] / FRED_CPIAUCNS[overlap_year]
  S1501-B[year] = FRED_CPIAUCNS[year] * scale_factor   for year in [2012, 2025]

SPLICE
  S1501_extended[year] = S1501_book[year]    for year in [1774, 2011]
  S1501_extended[year] = S1501-B[year]       for year in [2012, 2025]
```

Because both series are on the same 1982-84 base, `scale_factor` is expected to be ~1.000 (to within rounding); the explicit overlap-anchor calculation is retained for robustness against any future BLS rebase or revision.

## 4. Worked example

If `FRED_CPIAUCNS[2011] = 224.939` and `S1501_book[2011] = 224.94`, then `scale_factor = 224.94 / 224.939 = 1.000004`. Negligible adjustment.

## 5. Proxies

**None.** FRED `CPIAUCNS` is BLS CPI-U All Urban Consumers (NSA), the same series MeasuringWorth uses for its post-1913 segment. No concept substitution.

## 6. Synthetic data

**None permitted.** NaNs propagate.

## 7. Failure modes & graceful degradation

| Failure | Detection | Action |
|---|---|---|
| `FRED_API_KEY` not set | `S00_config` flags | Loader returns `{status: degraded, fred_status: missing_key}`; processor publishes 1774-2011 only; registry stamped `extension_status: api_key_missing` |
| FRED API non-200 | `S00_apis.fred_observations()` raises | Retries 3x then degrades |
| FRED returns NaN at 2011 | Processor walks back through 2010, 2009, ... 2005 | If all NaN, FAIL hard |

## 8. CD2 divergence pre-disclosure

CD2's S076 used MeasuringWorth as well; level values should match exactly at the four spot-check years (1774, 1899, 1949, 1999). The validator reports this informationally.
