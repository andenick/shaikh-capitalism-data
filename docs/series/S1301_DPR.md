## 1. Definition

Figure 13.7 (p. 633) is a purely illustrative schematic of equation (13.43):

```
ln Y_t = ln Y_0 + alpha * t + eta_t,   eta_t = sum_{i=1..t} epsilon_i
```

A deterministic linear trend (with slope `alpha`) plus a random-walk-with-
drift error term. The chart's x-axis is an unlabelled time index t in [0, 75]
(NOT calendar years); the y-axis is `ln Y` in model units 8.0 to 12.5.

## 2. Why it matters in Chapter 13

Fig 13.7 motivates Shaikh's classical view of output growth: actual output
fluctuates around an equilibrium path determined by net profitability and the
capacity-capital ratio (eqs. 13.41-13.42). The figure introduces the
visualization Shaikh re-uses for figures 13.8, 13.9, 13.10 with different
shock configurations.

## 3. Source

Shaikh 2016, equation (13.43), p. 632. No empirical data file backs this
figure; it is an analytical realisation.

## 4. Construction

`formula` (per eq. 13.43). Renders as a `theoretical` content_type series
