# S1002 — Extension Provenance Record

**Series**: S1002 — Interest Rate and Price Level
**Phase**: 6 (Extension)
**Construction classification**: `composite`
**Extension method**: same-agency continuation via FRED CSV endpoints

---

## 1. Why extendable

Both subseries' modern half is a continuously-published government / Federal Reserve series: Moody's Aaa via FRED `AAA`, BLS PPIACO via FRED `PPIACO`. The extension is a same-series continuation; the No-Lazy-Splices rule does not apply because the subseries are direct observations.

## 2. Method

```
INPUTS
  S1002-A_book   = USLR['iblong'] 1857-2011 (percent yield)
  AAA_raw        = FRED fredgraph.csv?id=AAA, monthly -> annual mean
  S1002-B_book   = USLR['USWPI']  1780-2011 (index 1947=100)
  PPIACO_raw     = FRED fredgraph.csv?id=PPIACO, monthly -> annual mean

YIELD EXTENSION (S1002-C)
  AAA is already in percent; for year > 2011, S1002-C[y] = AAA_annual_mean[y]

PRICE LEVEL EXTENSION (S1002-D)
  Reindex anchor: 2011 overlap
  scale = USWPI[2011] / PPIACO_annual_mean[2011]
  S1002-D[y] = PPIACO_annual_mean[y] * scale     for y > 2011
```

## 3. Proxies

**None.** Same agencies, same underlying series. `proxy: false` on all 4 subseries.

## 4. Synthetic data

**None.** If FRED returns no observations for a year (e.g. partial current year), that year is omitted.

## 5. Failure modes

| Failure | Action |
|---|---|
| FRED CSV non-200 | Loader retries 3× with backoff; on persistent failure, publishes book period only with `*_status: api_error` |
| 2011 overlap missing in either series | Walk back to 2010, then 2009; fail hard below 2005 |
| USLR workbook missing | Hard fail |

## 6. CD2 divergence

CD2 S052 used a similar splice; ~1% divergence expected from rounding and aggregation differences. Reported informationally only.
