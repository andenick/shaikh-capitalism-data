# S1103 — Extension Provenance Record

**Phase**: 6 (Extension)
**Construction**: `formula` (REER PPI / adjusted RULC ratio)
**Extension method**: re-compute formula with extended components (BIS PPI EER + OECD/Conference Board ULC)
**Status v1**: `not_attempted_v1` (deferred to Phase 9 enrichment)
**Proxy flag**: `true` (post-2009 components both substituted with concept-match)

---

## 1. Why this series is extendable (and why it requires coordinated proxy reconstruction)

S1103 is `formula`: published `rxrrulcratio1 = rxr1 / rulcadjratio1rescaled`.
Per anu-framework's No-Lazy-Splices rule, extension must re-compute the
ratio with extended numerator AND denominator series. Both inputs come from
the BLS International Labor Comparisons program, which was discontinued in
2013 — so both must be substituted via documented proxies.

## 2. Concept Match Justifications

### Numerator (REER PPI)
Inherits S1102's BIS PPI EER substitution and CMJ (see `S1102_EPR.md` §2).

### Denominator (adjusted RULC)
Two-variant strategy per Phase 4 CH11 chapter recommendation:

- **Open-license path (S1103_open)**: OECD ULC_QUA (whole-economy ISIC A-T).
  Concept gap acknowledged: OECD is whole-economy, Shaikh used
  manufacturing-only BLS ILC. Flagged `proxy: true` with sectoral-narrowing
  note.
- **High-fidelity path (S1103_hifi)**: Conference Board International Labor
  Comparisons (explicit successor program to BLS ILC). Direct concept match
  (manufacturing ULC). Requires Conference Board licensed access. NOT
  shippable in open-license replication package.

The CPI/PPI `tau` adjustment uses BLS/FRED CPI and World Bank WDI PPI which
remain active — no substitution.

## 3. Method (planned, v1 = not attempted)

```
Numerator (S1103-num) = BIS-extended rxr1  (see S1102 EPR)
Denominator (S1103-den) = (RULC_c / RULC*_c) * (tau_c / tau*_c)
    where RULC = ULC / CPI; tau = CPI / PPI; both for c (country) and *
    (trading partners weighted basket).
S1103[c, year>=2010] = S1103-num[c,year] / S1103-den[c,year]
Rescale denominator to share REER period average (same as Shaikh).
Splice to book values at 2009 with overlap-anchor to 1994-2009 sub-window.
```

## 4. No-Proxy compliance

Required: both proxy substitutions are flagged in the registry with their
respective CMJs. Two variants are emitted so users can choose the open vs
high-fidelity path.

## 5. No-Synthetic compliance

Any NaN in numerator OR denominator propagates to the ratio.

## 6. Failure-mode table

| Failure | Action |
|---|---|
| BIS unavailable | Numerator NaN — ratio NaN |
| OECD ULC missing for country | Denominator NaN — ratio NaN |
| Conference Board license absent | Skip S1103_hifi variant; emit S1103_open only |

## 7. CD2 divergence pre-disclosure

CD2's S062 was classified `direct` and lazily spliced; RSCD v1 reproduces
CD2's book-period values exactly (MAE 0.0). Phase 9 extension will introduce
a coordinated proxy-pair reconstruction not present in CD2.
