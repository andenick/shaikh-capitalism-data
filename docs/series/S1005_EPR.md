# S1005 — Extension Provenance Record

**Series**: S1005 — Dividend Yield vs Bond Yield
**Construction**: `composite`
**Extension**: same-series continuation via Shiller `ie_data.xls` + FRED CSV

---

## 1. Extendability

Yes. Shiller updates `ie_data.xls` continuously; FRED `GS10` and `AAA` are live.

## 2. Method

```
DIVIDEND YIELD (S1005-D dividend)
  Shiller ie_data.xls 'Data' sheet, monthly D and P columns
  Annual aggregation: mean of monthly (D/P*100) over each calendar year
  Append years > 2011 to S1005-A book values

10YR GOV YIELD (S1005-D gov)
  Prefer Shiller GS10 column from ie_data.xls (annual mean)
  Fallback to FRED fredgraph.csv?id=GS10
  Append years > 2011

CORPORATE YIELD
  Inherited from S1002 extension (FRED AAA)
```

## 3. Proxies

None — Shaikh's dividend yield IS Shiller's published series.

## 4. Synthetic data

None; FRED partial-year NaN propagates.

## 5. Failure modes

| Failure | Action |
|---|---|
| Shiller `ie_data.xls` HTTP non-200 | Use FRED `MULT` (Multpl mirror) — not implemented in Phase 5, degrade to book-period only |
| FRED GS10 unavailable | Use Shiller GS10 column only |
| Both unavailable | Publish book period only |
