## 1. Why this series is extendable

S1505 is `time_series + composite`; all components remain published. Extension recomputes the formulas — not a lazy splice.

## 2. Method

```
INPUTS (book period, authoritative)
  S1505_book = Appendix15_USInflation.xlsx cols pi, sigma, sigma', uL, uLintensity
               + Appendix14_InflationULdata.xlsx for BLS series cross-ref

INPUTS (extension period, re-fetched)
  BEA NIPA T1.1.4 GDP deflator
  BEA NIPA T1.16 row "Net operating surplus" (validate by label not line number)
  BEA NIPA T5.3.5 line 2 Nonresidential Private Fixed Investment (Igcorpbea)
  FRED UNRATE (BLS Civilian Unemployment Rate, percent) - normalize /100

FORMULAS
  pi_t      = ln(pGDP_t / pGDP_{t-1})
  sigma_t   = Igcorpbea_t / NOS_t
  sigma'_t  = HP-filter normalize sigma per Handfas 2012
  uL_t      = UNRATE_t / 100
```

## 3. No-proxy disclosure

**None.** BEA NIPA + BLS UNRATE are the same sources Shaikh used. FRED republishes BLS UNRATE without modification.

## 4. No-synthetic disclosure

**None.** Missing extension years are simply not appended.

## 5. Failure-mode table

| Failure | Detection | Action |
|---|---|---|
| `BEA_API_KEY` missing | `S00_config` | Book period only published |
| `FRED_API_KEY` missing | `S00_config` | uL extension skipped |
| BEA T1.16 row label changed | label-validation in loader | Warn + skip; document |
| Handfas 2012 sigma formula needs vintage adjustment | manual review | Documented in Phase 9 audit |

## 6. CD2 divergence pre-disclosure

CD2's S085/S088/S089 used similar BEA/BLS sources; level values expected to match within rounding.
