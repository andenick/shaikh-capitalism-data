# S105: Corporate Incremental Rate of Profit (IROP)

**Chapter 6: Capital and Profit** | **Figures**: Fig6.7 | **Period**: 1948–2024

## What This Measures

The incremental rate of profit (IROP) measures the return on new investment. Nominal adjusted IROP = delta_GOScorp / (IGcorpbea + delta_INVcorp) - 1, where delta is the first difference, GOS is gross operating surplus, IG is gross private fixed investment, and INV is inventories. The NIPA variant uses conventional corporate profits instead of adjusted GOS. IROP is inherently more volatile than the average rate of profit.
