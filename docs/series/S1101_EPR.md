# S1101 — Extension Provenance Record

**Phase**: 6 (Extension)
**Construction**: `composite`
**Extension method**: IMF DOTS (Direction of Trade Statistics) per-country X and M continuation; X/M recomputed.
**Status v1**: `not_attempted_v1` (deferred to Phase 9 enrichment)

---

## 1. Why this series is extendable

S1101 components are X and M in current USD published continuously by IMF for
all 15 countries (IFS and DOTS are both active). The X/M ratio is the
canonical published Shaikh figure; the same ratio recomputed from DOTS X and
M for post-2009 years is a direct continuation of the same observed series.

## 2. Classification

`composite`. The book series is a per-country pass-through of IMF IFS X and M;
extension fetches the same source for new years and computes X/M identically.
No lazy splice is needed — the X/M ratio is itself reconstructable from raw
flows year by year.

## 3. Method (planned, v1 = not attempted)

```
For each country c in {US, UK, DE, JP, ...}:
    Fetch DOTS_X[c, year] and DOTS_M[c, year] for year in [2010, current]
    S1101[c, year] = DOTS_X[c, year] / DOTS_M[c, year]
Belgium 1960-1992: preserve AMECO X+M backfill; X/M remains NaN.
```

## 4. No-Proxy disclosure

None. IMF DOTS publishes the same X and M flows IFS does for the same
countries; same agency, same concept.

## 5. No-Synthetic disclosure

None. NaN propagates if DOTS lacks a country/year.

## 6. Failure-mode table

| Failure | Action |
|---|---|
| IMF DOTS API down | Loader returns `extension_status: api_unavailable` |
| China data missing | S1101-E remains `cd2_addition` placeholder |
| Belgium pre-1993 X/M | NaN preserved, never imputed |

## 7. CD2 divergence pre-disclosure

CD2 S060 included China (S060-E); RSCD honors this with the `cd2_addition`
flag but does not emit S1101-E rows in v1. Other 15-country values match
verbatim.
