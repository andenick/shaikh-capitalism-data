# S1606 — Extension Provenance Record

**Series**: S1606 — Household Debt-Service Ratio
**Phase**: 6 (Extension)
**Construction classification**: `direct`
**Extension method**: direct continuation via FRED `TDSP` and `FODSP` (the modern public mirrors of the Fed legacy DDP series the book cites); emit both quarterly (primary) and annual-mean (secondary) variants
**Authored**: 2026-05-18
**Author**: opus-subagent-ch16-fanout

---

## 1. Why this series is extendable

`time_series`. The Federal Reserve continues to publish the Household Debt Service and Financial Obligations Ratios quarterly without interruption since 1980. FRED republishes the two ratios as `TDSP` (DSR) and `FODSP` (FOR). No methodology change since 1980.

## 2. Construction classification

`direct` — single-source per subseries, no algebraic combination. Lazy-splice prohibition does not apply because the extension uses the *exact same observed series* the book cites (Fed legacy DDP `FOR/FOR/DTFD%YPD.Q` -> FRED `TDSP`).

## 3. Method

```
INPUTS (each extension run, TTL=30d)
  FRED TDSP   quarterly SA, percent
  FRED FODSP  quarterly SA, percent

UNIT CONVERSION (FRED -> book scale)
  tdsp_decimal_t  = TDSP_t  / 100.0     # 11.1 -> 0.111
  fodsp_decimal_t = FODSP_t / 100.0

SPLICE (primary quarterly variant)
  S1606-A_extended[t] = book FOR[t]      for t in [1980Q1, 2012Q4]
  S1606-A_extended[t] = fodsp_decimal_t  for t in [2013Q1, 2024Q4]
  S1606-B_extended[t] = book DSR[t]      for t in [1980Q1, 2012Q4]
  S1606-B_extended[t] = tdsp_decimal_t   for t in [2013Q1, 2024Q4]

ANNUAL VARIANT (secondary)
  S1606-A_annual[year] = mean( S1606-A_extended[year-Q1..Q4] )
  S1606-B_annual[year] = mean( S1606-B_extended[year-Q1..Q4] )

OUTPUT
  Primary parquet (quarterly):  Technical/data/processed/S1606.parquet
                                  columns: year, quarter, qdate, value, subseries_id, source_id, units
  Secondary parquet (annual):   Technical/data/processed/S1606_annual.parquet
                                  columns: year, value, subseries_id, source_id, units
```

## 4. Component re-fetching

Every extension run re-pulls TDSP and FODSP from FRED at quarterly frequency. The annual variant is a deterministic average; recomputed from scratch each run (never persisted across runs without recompute).

## 5. Proxies

**None.** TDSP = the book's `DTFD%YPD.Q` identifier under FRED's modern naming. FODSP is the parallel FOR series cited in CD2 and tracked here. Both are direct Fed-published series. `proxy: false` on all subseries.

## 6. Synthetic data

**None permitted.** If FRED returns NaN for any quarter, the value is NaN in S1606. Annual variant tolerates up to 1 NaN quarter (averages over remaining); if 2 or more NaN, the annual value is NaN.

## 7. Failure modes & graceful degradation

| Failure | Detection | Action |
|---|---|---|
| FRED_API_KEY missing | startup | Extension skipped; book quarterly published only |
| TDSP / FODSP 5xx | retry 3× | Mark `extension_status: api_error`; publish book period only |
| Splice gap (book ends Q4 2012, FRED starts mid-year) | overlap check | Walk back to 2012Q1 / 2011Q4 anchor; if no overlap, fails hard |
| FRED unit drift (percent vs decimal) | sanity assertion `0.05 < median < 0.25` | If outside band, log warning and skip the suspect rows |
| FOR < DSR violation | post-merge assertion | Log year/quarter; do NOT auto-correct |

## 8. CD2 divergence

CD2's `S101` decomposed into A (FOR) and B (DSR). RSCD S1606 preserves this naming and adds extension subseries C/D plus the annual variant E. Values on the book period should match CD2 within 0.1% (both pull from the same Fed publication).

## 9. Splice diagnostics (V03)

- Per-quarter MAE on book period 1980Q1-2012Q4 vs Appendix 16.2 columns — expected ~0%
- Splice continuity 2012Q4 -> 2013Q1: FRED value should match book value within ±0.5%
- Sample quarters (per CD2 S101 validation): 1980Q1, 1985Q1, 2000Q1, 2007Q1, 2012Q1
- FOR > DSR ordering: assertion at every quarter
- Annual variant validation: exact arithmetic mean of corresponding quarters
