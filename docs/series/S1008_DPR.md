## 1. Definition

**S1008** is Shaikh's central empirical comparison in Chapter 10 (Figure 10.10):

- `preq` — real S&P composite price (Shiller P deflated by BEA gross investment deflator)
- `prstarshiller1` — Shiller's P* (PV of real dividends at constant r) re-deflated by BEA pIGcorpbea
- `prweq` — *warranted* real equity price computed via eq. (10.31):
  ```
  prweq_t = prweq_{t-1} * (1 + rI_t) - dvr_t
  with prweq[1947] = preq[1947] + adj
  ```
  Footnote 24 describes the original calibration of `adj` as a same-mean condition over [1948, 2011]. The workbook's published prweq column (Appendix10_IntroPPrice col 33) uses `adj = 6.75` as a fixed parameter (per the cell-header note "initial value = real peq + adj parameter (6.75)"; verified by `prweq[1947] − preq[1947] = 72.052 − 65.302 = 6.75` exactly). RSCD pins `adj = 6.75` to reproduce the book column exactly. This is consistent with the Phase 4 "must not re-anchor on extended interval" rule (re-solving the mean-match on any longer panel would shift adj and alter every historical value).

## 2. Why it matters

This is Shaikh's "the equity market follows the warranted price in the long run" empirical demonstration — the capstone of Chapter 10.

## 4. Construction

```
preq[t]            = read pass-through from IntroPPrice col 25
prstarshiller1[t]  = read pass-through from IntroPPrice col 38
                     (where present; col is NaN after 2009)
prweq:
  init: prweq[1947] = preq[1947] + adj
        where adj is chosen so mean(prweq[1948..2011]) = mean(preq[1948..2011])  (footnote 24)
  recur: prweq[t] = prweq[t-1] * (1 + rI[t]) - dvr[t]
         where rI = S1007-B (iropcorp adjusted)
               dvr = Shiller D / pIGcorpbea, from IntroPPrice col 26
```
