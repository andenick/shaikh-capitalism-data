# S1007 — Extension Provenance Record

**Series**: S1007 — Equity Rate vs Corporate IROP
**Construction**: `composite` (three derived rates from BEA + Shiller components)
**Extension**: deferred to Phase 6 (NIPA vintage open question)

---

## 1. Extendability

Yes in principle: BEA NIPA (NOS, gross investment, deflators) and Shiller (price, dividend) are live. The blocker is methodological: the book's pre-2018 NIPA vintage and Shaikh's "Monetary Interest Paid by Nonfinancial Sector" adjustment require explicit table-line pinning before the extended panel can be assembled with fidelity to the book.

## 2. Recommended Phase 6 method (NOT executed in Phase 5)

```
INPUTS (post-2011 extension)
  P_shiller[t], D_shiller[t]              from Shiller ie_data.xls annual
  CPI_shiller[t]                          from Shiller ie_data.xls annual
  pIGcorpbea[t] = FRED A006RD3A086NBEA    (gross investment deflator, 2017=100; reindex to book base)
  NOS_nfcorp[t] = FRED W260RC1A027NBEA    (nonfinancial corporate NOS, billions current $)
  MIP_nfcorp[t] = BEA NIPA Table 7.11 line ?? (MUST be pinned)
  IG_real[t]    = BEA NIPA Table 5.2.3 real gross investment
  INV_real[t]   = BEA NIPA Table 5.6.3 real change in private inventories

DERIVED
  preq[t] = P_shiller[t] / pIGcorpbea_reindexed[t]
  dvr[t]  = D_shiller[t] / pIGcorpbea_reindexed[t]
  rreq[t]    = (preq[t] - preq[t-1] + dvr[t]) / preq[t-1]
  NOS_adj[t] = NOS_nfcorp[t] + MIP_nfcorp[t]    # undo NIPA interest-as-cost convention
  iropcorp[t]     = (NOS_adj_real[t] - NOS_adj_real[t-1]) / (IG_real[t-1] + INV_real[t-1])
  iropcorpnipa[t] = (NIPA_gross_profits_real[t] - NIPA_gross_profits_real[t-1]) / IG_real[t-1]

DISCLOSURE
  - Vintage stamp: record the BEA vintage actually pulled
  - Concept Match Justification for any line whose definition shifted in 2018 comprehensive revision
```
