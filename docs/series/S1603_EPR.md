## 1. Why this series is extendable

`time_series`. The US 3-month T-bill (`TB3MS`) is continuously published. The OECD weighted aggregate is reconstructable from the same underlying inputs (Fed H.10 trade weights + IMF IFS country rates). An off-the-shelf approximation (OECD MEI) is available for users who don't need book-fidelity replication.

## 2. Construction classification

`composite` — multiple country rates weighted into one aggregate. The Anu lazy-splice rule applies: the extension must recompute the weighted aggregate from extended component data, not naively splice a growth rate onto the book-period level. This is what `S1603_replicated` does.

## 3. Method — Variant 1: S1603_replicated (canonical)

```
INPUTS
  Fed H.10 weights:  https://www.federalreserve.gov/releases/h10/Weights/
    -> table_a.csv (broad index trade weights, latest vintage)
  IMF IFS country money market rates:
    -> https://data.imf.org/IFS (FM_PA / FIMM_PA / Money Market Rate)
  Shaikh OECD basket (per CD2 S098 country list):
    USA omitted (US is plotted separately), plus
    DEU, FRA, GBR, JPN, ITA, CAN, AUS, CHE, SWE, NLD, BEL, KOR, MEX, ESP,
    plus additional partners weighted into Fed H.10 broad index.

AGGREGATION
  For year t in [1960, 2024]:
    weights w_{c,t}  = latest-vintage Fed H.10 weight for c (annual avg of monthly)
    rates   i_{c,t}  = IMF IFS Money Market Rate for c (annual avg of monthly)
    ioecd_t          = sum_c (w_{c,t} * i_{c,t}) / sum_c (w_{c,t})

SPLICE
  S1603-B_book        = Appendix 16.2 OECD column (1960-2012)
  S1603-F_replicated  = computed ioecd_t from Fed H.10 + IMF IFS (1960-2024)
  Cross-check 1960-2012: replicated should match book within +/- 50bp (validates rebuild)
  Final published S1603-B_extended: book 1960-2012 || replicated 2013-2024
```

## 4. Method — Variant 2: S1603_oecd_mei (proxy:true, low-effort)

```
INPUTS
  OECD MEI:  https://data-explorer.oecd.org/  ->  Subject = IRSTCI01 (Short-term interest rates)
             Frequency = M, aggregated to annual avg

S1603-E[t] = OECD_IRSTCI01[t]  for t in [1960, 2024]

CONCEPT MATCH JUSTIFICATION
  Both S1603-F (replicated) and S1603-E (OECD MEI) measure trade-weighted OECD
  short-term interest rates. They differ in:
    (a) weighting base: Fed H.10 trade weights (Shaikh-Ragab) vs OECD GDP-PPP weights
    (b) basket composition: minor differences in country list and tier coverage
  The two should track within +/- 100bp in any given year. OECD MEI is offered
  as a low-friction alternative for users who do not require book-fidelity
  replication. Flagged `proxy: true` with explicit CMJ.
```

## 5. US component extension

```
S1603-A_book     = Appendix 16.2 US column (1960-2012)
S1603-D[t]       = FRED TB3MS annual-avg[t]  for t in [2013, 2024]
S1603-A_extended = book 1960-2012 || FRED 2013-2024 (direct splice, same series)
```

Same-agency direct splice; no proxy concern.

## 6. Proxies

- **S1603_oecd_mei** is flagged `proxy: true` with CMJ above.
- **S1603_replicated** is NOT a proxy — it rebuilds Shaikh-Ragab's exact methodology using the original inputs.
- US (TB3MS): same series, same agency, direct continuation. `proxy: false`.

## 7. Synthetic data

**None permitted.** If IMF IFS or Fed H.10 weights are unavailable for a country/year, that country is dropped from the weighted sum for that year, the remaining weights re-normalized, and the diagnostic flags the country/year omission. No fabricated values, no carry-forward.

## 8. Failure modes & graceful degradation

| Failure | Detection | Action |
|---|---|---|
| FRED_API_KEY missing | startup | US extension marked unavailable; book US 1960-2012 published only |
| Fed H.10 weights CSV unreachable | HTTP non-200 | Replicated variant marked unavailable; OECD MEI variant remains valid |
| IMF IFS country gap | per-country fetch | Drop country from sum that year; re-normalize; log in diagnostics |
| OECD MEI endpoint reorganized | HTTP non-200 | OECD MEI variant marked unavailable; replicated variant remains canonical |
| Both variants fail | composite check | Book period 1960-2012 published; extension marked `data_unavailable` |

## 9. CD2 divergence

CD2's `S098` had 5 subseries (A OECD annual, B US from Appendix 6.8, C OECD MEI alt, D OECD/EU ratio, E combined fill-in). RSCD S1603 retains A (book OECD), B (book US), C (book EU), plus three new extension subseries (D US-TB3MS, E OECD-MEI, F replicated). The fill-in logic in CD2's S098-E is absorbed into the replicated variant's per-country drop-and-renormalize behavior.
