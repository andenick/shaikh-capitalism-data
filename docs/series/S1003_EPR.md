# S1003 — Extension Provenance Record

**Series**: S1003 — Relative Price of Finance
**Construction classification**: `formula` (derived)
**Extension method**: recompute formula from extended components (No-Lazy-Splices)

---

## 1. Extendability

Yes — the underlying components (`iblong` → FRED AAA; `USWPI` → FRED PPIACO reindexed to 1947=100) are both extendable. The ratio is recomputable through the most recent annual observation.

## 2. Method

```
For each year y >= 2012:
  i[y] = S1002 extended yield   (FRED AAA annual mean)
  p[y] = S1002 extended PPI     (FRED PPIACO annual mean * scale, scale = USWPI[2011]/PPIACO[2011])
  S1003[y] = (i[y] / i[1947]) / (p[y] / p[1947])

i[1947] and p[1947] are fixed at book-period values (read from Appendix10_USLR.xlsx).
```

## 3. No lazy splice

Per the Anu rule: this is a derived (formula) quantity; we never splice the published `ib/p` ratio. We re-fetch the components and recompute the ratio. This preserves analytical meaning across the splice boundary.

## 4. Proxies

**None.** AAA and PPIACO are direct continuations of the components Shaikh used.

## 5. Failure modes

If S1002 components are unavailable for extension, S1003 publishes book period only with `extension_status: components_unavailable`.

## 6. CD2 divergence

Informational only.
