## 1. Definition

Trade-weighted multilateral real effective exchange rate, PPI-deflated, for
the United States (S1102-B) and Japan (S1102-A), annual 1960-2009. Index base
2002 = 100. Appears in Shaikh (2016) as **Figure 11.3** (p. 526).

Shaikh's REER follows eq. (11.16):
`REER_c = (ppi_c * e_c) / ppi*_c`
where `ppi*_c` is the geometric mean of trading-partner PPIs weighted by
`w_j = (X_j + M_j) / sum(X+M)`.

## 2. Why it matters in Chapter 11

Figure 11.3 establishes that REERs are NOT stationary in either the short or
long run — directly rejecting strong-form PPP. This motivates Shaikh's
search for non-PPP fundamentals (relative real unit labour costs, S1103).

## 4. Construction

`formula` (per eq. 11.16). For the book period, however, the published rxr1
column IS the canonical author-computed value; no re-derivation is required
within 1960-2009. Re-derivation is required only for the post-2009 extension.

## 7. Caveats

1. **BLS ILC discontinued 2013.** Original primary source (Tables 9 and 11) is
   no longer published. Phase 4 canonicalized BIS PPI EER broad index as
   substitute; see EPR.
2. **Sample basket.** Shaikh's REER uses a static 19-country basket; BIS uses
   a rolling ~60-country basket — concept match documented in EPR.
3. **Original URL https://www.bls.gov/fls/ichcc.htm returns 403.** Use
   archival landing https://www.bls.gov/ilc/ per Phase 4 url_substitutions.
